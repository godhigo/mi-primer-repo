¿Qué es la Realidad Virtual?
La realidad virtual es una tecnología que te pone en un entorno digital mediante un visor o gafas especiales. Al usarlas, puedes ver y explorar un mundo creado por computadora como si estuvieras realmente dentro de él, permitiéndote moverte y mirar a tu alrededor en 360 grados.
Repositorio
Un repositorio (o repo) es como una carpeta en línea donde guardas todo el código y archivos de un proyecto de software. En GitHub (o Git), además de guardar el código, también te permite ver el historial de todos los cambios que se han hecho, para que puedas seguir cada modificación y ver quién hizo qué en el proyecto.
Git add
Git commit
Git push
Este comando es como el primer paso para preparar los cambios que queremos hacer. Imagina que seleccionas los archivos que vas a enviar como tarea o que quieres guardar en una versión específica de tu proyecto. No los has subido aún, pero ya le estás diciendo a Git: "Estos son los cambios que quiero tener en cuenta para después".
Aquí es cuando Git guarda oficialmente esos cambios que seleccionaste. Es como un "checkpoint" en tu proyecto: cada vez que haces un commit, Git guarda una versión exacta de los archivos para que puedas regresar si es necesario. Con el -m` le agregas un mensaje corto que explica qué cambios hiciste, algo así como dejar una nota para que en el futuro te acuerdes de por qué hiciste ese commit.
Finalmente, este comando es el que sube tus cambios al repositorio remoto, es decir, lo manda a la nube. Así que después de hacer commit, con el push le dices a Git: "Ok, sube todo lo que he hecho a GitHub". Es básicamente cuando mandas tu trabajo a un servidor remoto para que esté seguro y accesible desde cualquier lugar.
GitClone
Es un comando en Git y lo puedes usar de dos maneras. La primera es clonar usando el comando git clone seguido de la URL del repositorio para descargar una copia completa del proyecto, incluyendo todos los archivos y el historial de cambios. La segunda opción es clonar un repositorio local, donde en lugar de una URL, proporcionas la ruta del repositorio en tu computadora o red local, y con el mismo comando creas una copia exacta del proyecto en otra carpeta de tu sistema.
