GitHub y Comandos Básicos

Aprende qué es GitHub, cómo funciona un repositorio y los comandos básicos de Git.

git add

Añade archivos al área de preparación antes de un commit.

git commit

Guarda los cambios realizados en el proyecto con un mensaje descriptivo.

git push

Sube los cambios locales a un repositorio remoto.

git clone

Copia un repositorio remoto en tu máquina local.
